# Feed Reader App

Aplicativo para a prova da Electric Dreams

## Instalação

Abrir o git Bash em alguma pasta e siga esses passos para a instalação

```bash 
git clone https://github.com/Eskimotick/feed-reader-app.git
cd feed-reader-app/
npm install
```

## Usage

Para rodar localmente

```bash
ionic serve
```
